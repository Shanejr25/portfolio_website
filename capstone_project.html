DragonDen Mobile App

Client

DragonChain

Category

Mobile App

Project Date

March 02, 2019

Project Link

The DragonDen Mobile App was my senior project at Central Washington University. I worked in a team of 5 and created an application similar to Reddit, based on Dragonchain's web application. We wrote this in the Flutter framework and Dart language to make it easy to run on both Android and IOS without having to develop multiple times for each platform. The application makes JSON calls to their API and allows you to create and comment on text/image posts while utilizing the user's Dragon currency.

Note: Not much code is included on the project link page because of an NDA.
